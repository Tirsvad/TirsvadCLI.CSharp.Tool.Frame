# Frame handler
Frame handler. Create a frame with text inside. Frame can be centered to windows.

## How to get started

Nuget package: https://www.nuget.org/packages/TirsvadCLI.Frame/

### Installation

```sh
dotnet add package TirsvadCLI.Frame
```
